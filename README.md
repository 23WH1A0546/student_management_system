# Student Management System (MERN Stack)

A simple Student Management System developed using the MERN stack with role-based access for Admin and Teacher users.

## Description
This project allows Admins to manage student records and Teachers to view student details, mark attendance, and update marks.  
It extends basic CRUD functionality by adding real-world features like attendance tracking and automatic grade calculation.

## Features
- Add, view, edit, and delete students (Admin)
- View student list and details (Admin & Teacher)
- Mark student attendance (Present / Absent)
- Update marks with automatic grade calculation
- Search students by name, roll number, or branch

## Users
- **Admin:** Full access to student management
- **Teacher:** View students, mark attendance, update marks

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
